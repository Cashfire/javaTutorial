## Contents
JAVA 基础
java基础部分一共分为如下几大块进行讲解：
一、java的基础部分，该部分仅仅只讲解了相应的语法和操作，依然由于我们的学生的基础，所以该部分非常的罗嗦和缓慢，如果你是零基础可以考虑看看，有编程基础，完全可以跳过该部分的学习。
二、面向对象部分，该部分主要分析面向对象的基础，并且基于一个游戏的小例子来逐步讲解面向对象中的各个概念，该部分的教学重点是面向对象，如果你对面向对象已经掌握也不用费心下载这部分的视频了。但是这个小游戏的例子会贯穿整个教学，并且在最后的项目中会基于该游戏实现一个基于网络版的小游戏项目，该部分如果不想看视频，可以考虑看看源码即可。
三、Java的专题部分：该部分一共分成如下几部分进行讲解
·Java的常用类：该部分主要介绍如何使用doc文档，和一些小例子的练习
·Java的数组和集合类：该部分主要介绍数组和java的集合类，该部分引入了一个用户管理的小例子
·Java的异常处理：该部分以java的异常处理为重点，该部分扩展用户的小例子
·Java的IO：该部分主要讲解了输入输出，并且扩展了小游戏的例子（该部分没有涉及NIO）
·Java的GUI：图形用户界面的讲解，主要讲解的布局和绘图，并且简单扩展了小游戏的例子
·Java的多线程：Java如何处理多线程，如何同步线程，如果让线程之间进行通信（没有讲线程池和线程组，这些概念完全可以自学）
·Java的网络编程：主要通过一个聊天的小程序完成该部分的所有内容
·Java的正则表达式
四、j2SE的项目（该部分正则录制中，当讲完之后会及时更新）
·基于员工和部门管理的小例子（该部分已经更新，是通过xml进行数据存储，并且通过dom4j处理xml）
·完整实现游戏的小例子（单机版和网络版）
之后会根据教学逐步更新剩余内容。
五、更新相应的Java Web部分的视频，这次上传了jsp部分的视频，该部分的视频主要有如下几个：
1、JSP部分
2、Mybatis和MVC
3、SSH
六、更新了HTML+css+JS部分的内容
七、开始更新CMS项目的内容
计划：首先更新springMVC，其次更新svn+maven，完整的cms项目,最后更新lucene的全文索引
